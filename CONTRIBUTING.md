# Contributing

Thanks for improving Game Feel Lab.

## Local Development

The app is static and dependency-free. You can open `index.html` directly or run:

```bash
python -m http.server 8000
```

## Pull Request Checklist

- Keep controls usable on desktop and narrow screens.
- Keep exported JSON backward compatible when possible.
- Update the README if controls, presets, or export fields change.
- Avoid adding build tooling unless it unlocks a clear workflow improvement.
